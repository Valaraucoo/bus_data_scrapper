### Bus delay scrapper - Cracow

- Create and activate virtual enviroment:
    ```bash
    python3 -m venv venv
  
    # for Windows
    venv\Scripts\activate.bat
    
    # for Unix or MacOS
    source venv/bin/activate
    ```
- Set up project:
    ```bash
    python3 setup.py
    ```
- If everything is OK run `start.py`.
- Collected data will be stored in `data/` directory.
